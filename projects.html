Projects I've Worked on

SPOTIFY

Spotify, built using React, leverages a component-based architecture for a seamless user experience. Tailwind CSS ensures efficient and responsive design styling, while Redux handles state management for consistent and predictable application state. React Router provides smooth navigation for a user-friendly music streaming experience.

REAL-TIME CHAT

It provides a seamless, interactive platform for real-time messaging with a modern, intuitive user interface, allowing users to chat effortlessly across devices. The tech stack includes React for the front end, Firebase for backend services, and Zustand for state management.

PIG GAME

PIG game developed using JavaScript to provide an engaging, interactive experience with dynamic UI updates, real-time score tracking, and turn-based logic. The game's objective is for players to accumulate points by rolling a die, aiming to reach a target score while avoiding the risk of rolling a 1, which ends their turn with no points.

PASSWORD GENERATOR

The Password Generator project, built using React and styled with Tailwind CSS, provides a user-friendly interface for creating secure, random passwords enabling quick and responsive UI development. Users can customize password length and character sets, demonstrating dynamic state management and real-time updates in a modern web application.
